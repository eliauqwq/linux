# 高效文件编辑器——vim
vim(Vi IMproved)是一个高效的文件编辑器，是vi的加强版本，接下来我们来了解其基本知识。
## vim的工作模式
### 1.普通模式
该模式是vim编辑器的初始模式，在此模式下我们可以通过键盘上的方向键进行光标的上下左右移动，也可以根据以下的对应关系进行移动：

- `h`：向左移动
- `j`：向右移动
- `k`：向上移动
- `l`：向右移动

除此之外，该模式支持一些简单的操作，例如删除、复制、粘贴等等。
### 2.插入模式
如果需要复杂的文件编辑功能，我们可以在**普通模式**下使用以下按键进入**插入模式**：
- `i`：在光标所在的位置插入
- `a`：在光标位置之后插入
- `o`：在光标所在行下一行进入

在该模式下，只能使用方向键移动光标。

### 3.命令模式
该模式通常是输入对vim的一些内置命令，例如退出vim并且保存文件内容，或者替换文件中的模式。该模式在**普通模式**的基础上输入“：”即可进入。

需要注意的是，普通模式是另外两种模式的中间形态，我们在二者之间切换必须先切换到普通模式，一般我们按`esc`键回到普通模式。

## vim的一些基本操作
### 基本的使用方法
- 打开文件：`vim` `filename`，如果filename不存在，则自动生成一个名为filename的文件。
- 退出&&保存：在命令模式下，使用`w`进行保存，并使用`q`进行退出，`!`进行强制退出。
### 删除
>在***普通模式***下，删除等基本命令如下：
- `nx`：删除光标所在的位置及其后面的几个字符
- `ndd`：删除光标所在位置行及其后面几行
- `nd$`：删除从光标处到n行末尾
- `nd0`：删除从光标处到n行开头
- `ncc`：删除光标所在位置行及其后面几行，然后进入插入模式
### 复制
>在***普通模式下***，复制基本操作如下：
- `yy`：整行复制
- `y$`：从光标到行末尾
### 粘贴
>在***普通模式下***，粘贴操作如下：
- `p`：在光标下面粘贴
- `P`：在光标上面粘贴
### 替换
>在***命令模式***下，替换的语法格式为：
- `[范围]` `s` `/被替换内容/替换内容/` `[g]`

解读：
1. 范围默认是当前行，还有以下选项
- `n,m`：从n到m行
- `%`：整个文件
- '<'>观察模式选中的范围
2. `g`，整行替换，如果不加则只替换每行的第一个匹配到的对象。
### 查找
>***普通模式***下，查找的基本语法如下：
- `/` `pattern`：从光标处往后找
- `?` `pattern`：从光标处往前找

同时需要注意，上下查找匹配到的目标时：
- `n`：查看下一个匹配对象。
- `N`：查看上一个匹配对象。
